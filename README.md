## CÁCH HOẠT ĐỘNG

1. Tải và cài đặt <a href="https://getcomposer.org/">Composer</a>

2. Tải và cài đặt <a href="https://laravel.com/">Laravel</a>

3. Tải và cài đặt <a href="https://www.postgresql.org/">PostgreSQL</a>

4. Tải tệp "webcoffee" từ github repo này, giải nén toàn bộ thư mục

5. Tải file database: webcoffee.sql

6. Mở Visual Studio Code, chạy terminal. Hoặc mở cmd trên máy, chạy lệnh: cd webcoffee - php artisan serv

7. Past URL trang web được hiển thị lên trình duyệt: http://127.0.0.1:8000

10. Để truy cập trang quản lý, sử dụng link tài khoản sau:
    
   `http://127.0.0.1:8000/admin/login`
